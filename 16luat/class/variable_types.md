# Bảng Quy Định Biến Trong Hệ Thống Tam Giác

## 1. Biến Cạnh (Chữ thường)
| Biến | Ý nghĩa |
|------|---------|
| a, b, c | Các cạnh của tam giác |
| ma, mb, mc | Các trung tuyến |
| ha, hb, hc | Các đường cao |
| p | Nửa chu vi |
| P | Chu vi |
| S | Diện tích |
| r | Bán kính nội tiếp |
| R | Bán kính ngoại tiếp |

## 2. Biến Góc (Chữ HOA)
| Biến | Ý nghĩa |
|------|---------|
| A | Góc A (đối diện cạnh a) |
| B | Góc B (đối diện cạnh b) |
| C | Góc C (đối diện cạnh c) |

## 3. Quy Tắc Phân Biệt
- **Chữ thường** = cạnh, đường cao, trung tuyến, bán kính, diện tích
- **Chữ HOA** = góc

## 4. Ví Dụ
- `a, b` → Tìm cạnh c: Dùng luật có vế phải = `c` (chữ thường)
- `A, B` → Tìm góc C: Dùng luật có vế phải = `C` (chữ HOA)

## 5. Ưu Tiên Khi Suy Diễn
Khi có nhiều luật cùng vế phải, ưu tiên:
1. Luật có cùng TYPE (cạnh vs cạnh, góc vs góc)
2. Luật có ít premise nhất
3. Luật có ID nhỏ nhất
